# Manuels d’utilisation et de formation COYA

Ce répertoire regroupe **un document principal par module** (manuel unique pour les équipes terrain et la montée en compétence).

## Convention

- Chaque module a un sous-dossier : `docs/manuels-et-formation/<module>/`.
- Le fichier canon pour le module : **`MANUEL-UTILISATION-ET-FORMATION-v0.md`** (ou version ultérieure `v1`, `v2`… une fois la validation produit terminée).
- **v0** = état actuel du produit + méthodologies d’usage ; à remplacer ou enrichir par **v1** lorsque toutes les fonctionnalités prévues seront implantées et validées.
